Usage¶

fetch.calculate_time()¶

Calculates the time left until 20:03 (until Alternative updates), returns seconds

async fetch.daily_sentiment()¶

Fetches sentiment data every day at 20:03

fetch.delay()¶

Time to wait in seconds when starting the program, prevents conflicts with daily_sentiment()

async fetch.fetch_bitcoin()¶

Fetches bitcoin data and writes it to the database. Values will be set to “None” in case of failure and the user will be notified.

async fetch.fetch_data_with_retry(url, headers, max_retries=5, wait_time=10)¶

Main function responsible for fetching data. Includes a retry mechanism to handle errors and returns a parsed JSON response as a dictionary.

async fetch.fetch_eth()¶

Fetches eth data and writes it to the database. Values will be set to “None” in case of failure and the user will be notified.

async fetch.fetch_marketdata()¶

Fetches market data and writes it to the database. Values will be set to “None” in case of failure and the user will be notified.

async fetch.fetch_sentiment()¶

Function that fetches sentiment data and stores it globally. fng_name and fng_value will be set to “None” in case of failure and the user will be notified.

async fetch.fetch_stack()¶

Concurrently fetches data on btc, eth and the market

async fetch.hourly_sentiment()¶

Desynchronizes itself from the fetching functions and fetches sentiment data every hours.

async fetch.main()¶

Gathers all tasks in a concurrent main event loop, delays if necessary
