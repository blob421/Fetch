Getting started
================

Required dependencies
---------------------

 - Fetch
 - Python 3.x
 - crypto_data.sqlite 

Usage
------

 Run the script from the command line and follow the on-screen instructions:

 1. $ python fill_null.py
 2. Select an option ranging from 1 to 5.
 3. Choose a row to correct.
 4. Confirm the update.
 5. Exit
